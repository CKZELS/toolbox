写作指南

只要在固定目录里新增一篇 Markdown，就能生成首页卡片和详情页。

添加新工具

1
复制模板

在 content/tools 下新建一个英文文件名的 .md 文件。

2
填写信息

保持顶部字段完整，正文按普通 Markdown 写标题、列表和段落。

3
提交发布

推送到仓库后，静态平台会自动构建并更新网站。

固定格式

每篇工具文章都需要以下格式。顶部的字段会用于首页卡片、分类模块和详情页按钮。

---
id: example-tool
name: Example Tool
category: 效率工具
summary: 一句话说明这个工具解决什么问题。
icon: ✦
color: "#4B3FE3"
homepage: https://example.com/
order: 100
tags:
  - 标签一
  - 标签二
---

# Example Tool 使用心得

这里开始写正文。可以使用二级标题、列表、引用和代码块。

部署建议

推荐使用 Cloudflare Pages。它免费、简单、适合静态网站，连接 GitHub 仓库后只需要设置构建命令为 npm run build，输出目录为 dist。

适合无代码用户的维护方式

可以直接在 GitHub 网页里点开 content/tools，复制一篇已有文章，改字段和正文后提交。平台会自动重新生成网站。
