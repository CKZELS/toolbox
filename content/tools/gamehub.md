# GameHub 使用心得

GameHub 是一个让 Mac 用户更方便运行 Windows 游戏的工具。它把游戏库、兼容层、运行环境和启动配置放到一个更接近游戏启动器的界面里。

## 核心特点

- 支持管理和启动 Steam、GOG、Epic 等来源的游戏
- 围绕 Wine、Proton、GPTK 等兼容技术做封装
- 可以为不同游戏准备独立运行环境
- 比手动配置命令行和容器更适合普通玩家上手

## 适合什么场景

适合想在 Mac 上尝试 Windows 游戏，但不想自己折腾 Wine、Proton、DXVK、GPTK 和各种运行参数的人。它不能保证所有游戏都顺畅运行，但能把试错门槛降下来。

## 注意点

Mac 跑 Windows 游戏仍然有兼容性边界。反作弊、DRM、图形 API、输入设备和性能都会影响体验。建议把它当作“让更多游戏有机会运行”的工具，而不是当作稳定替代 Windows 游戏电脑。
