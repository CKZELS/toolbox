# Figma 设计指南

Figma 已经成为 UI/UX 设计里非常常见的工具。它最重要的价值不是“能画图”，而是让设计、产品、开发能在同一个文件里对齐。

## 核心优势

- 实时协作，适合团队一起看稿和改稿
- Auto Layout 可以更接近真实前端布局
- Components 和 Variants 适合沉淀设计系统

## 推荐用法

把常用组件先做成规范，再开始堆页面。这样后面改按钮、颜色、间距时，维护成本会低很多。
