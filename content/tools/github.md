# GitHub 使用技巧

GitHub 不只是放代码的地方。对于个人网站来说，它还可以负责版本管理、协作修改和自动部署。

## 实用功能

- Pull Request：在发布前检查改动
- Actions：自动构建和部署
- Pages：免费托管静态网站
- Issues：记录后续想做的功能

## 适合本项目的用法

把内容文章放在仓库里，修改 Markdown 后由平台自动生成网站。这样非技术用户也能通过网页编辑文件。
