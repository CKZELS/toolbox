# GitHub 使用技巧

GitHub 不只是放代码的地方。对于个人网站来说，它还可以负责版本管理、协作修改和自动部署。

## 实用功能

- Pull Request：在发布前检查改动
- Actions：自动构建和部署
- Pages：免费托管静态网站
- Issues：记录后续想做的功能

## 适合个人项目的用法

用 GitHub 管理个人网站、知识库或作品集很方便。它能记录每次修改，也能让你在发布前清楚看到变更。
