# Cherry Studio 使用心得

Cherry Studio 是一个面向日常使用的大模型桌面客户端。它适合把不同模型服务、助手模板、文件处理和对话工作流放到同一个界面里管理。

## 核心特点

- 支持 OpenAI、Gemini、Anthropic 等云端模型服务
- 可以连接 Ollama、LM Studio 等本地模型服务
- 内置大量预设助手，也可以创建自己的助手
- 支持文本、图片、Office、PDF 等材料处理
- 支持 Windows、macOS 和 Linux

## 适合什么人

适合经常在多个模型之间切换的人。比如有时用云端模型做高质量写作，有时用本地模型处理私密材料，有时又想用不同助手模板完成翻译、总结、代码解释或资料整理。

## 使用建议

先配置一两个最常用模型，不要一开始塞满所有服务。等常用场景稳定后，再慢慢补助手、知识材料和自动化能力。
