# Otty 使用心得

Otty 是一个现代终端应用。它不像传统终端那样只关注“能不能跑命令”，而是更在意敲击手感、视觉体验、会话组织，以及现在越来越常见的 AI 编程 Agent 工作流。

## 核心特点

- 原生、GPU 加速，滚动和渲染体验更现代
- 支持标签页、分屏、会话恢复和快速跳转
- 支持丰富主题、真彩色、连字、Unicode 和内联图片
- 对 Claude Code、Codex、opencode 等 Agent 工作流更友好
- 可以追踪长时间运行的命令，并在结束时提醒

## 适合什么场景

适合经常在终端里写代码、跑脚本、开本地服务和使用代码 Agent 的人。如果你的终端里同时有开发服务器、Git、SSH、AI Agent、日志窗口，Otty 的会话组织会很有价值。

## 使用建议

先把它当作主力终端试一周，重点观察分屏、会话恢复和 Agent 支持是否真的改善工作流。终端是高频工具，手感和稳定性比功能数量更重要。
