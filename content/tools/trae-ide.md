# Trae IDE 使用心得

Trae IDE 是面向 AI 编程的新一代开发环境。它基于 Code OSS，也就是 VS Code 的开源基础，因此整体交互会让 VS Code 用户比较熟悉。

## 核心特点

- 内置 AI 对话和代码辅助能力
- 可以结合项目上下文理解代码
- 适合用自然语言让 AI 解释、修改、生成和重构代码
- 对 VS Code 用户迁移成本较低

## 适合什么场景

适合想尝试 AI 编程，但又不想完全离开传统编辑器习惯的人。对于新项目搭建、代码解释、小功能修改、报错排查，它可以明显降低进入成本。

## 使用建议

把它当作“能读项目的编程搭档”，不要只让它补全单行代码。更好的用法是给它明确目标、约束和验收方式，让它围绕整个文件或整个功能协助修改。
