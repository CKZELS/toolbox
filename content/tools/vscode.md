# VS Code 使用心得

Visual Studio Code 是我日常开发中使用频率最高的工具。它轻量、快速，同时拥有成熟的插件生态。

## 适合谁

- 需要一个启动快、扩展多的代码编辑器
- 经常在不同语言和项目之间切换
- 希望把 Git、终端、调试都放在一个工作界面里

## 常用插件

- `ESLint`：JavaScript / TypeScript 代码检查
- `Prettier`：统一代码格式
- `GitLens`：增强 Git 历史和 blame 信息

## 小建议

先掌握命令面板和快速打开文件，再慢慢补插件。编辑器越轻，工作流越顺。
